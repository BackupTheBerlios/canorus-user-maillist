Good evening!

I released 0.4 Release Candidate a few minutes ago. Changes till beta
release:
- Ability to change the order of elements in element properties dialog
- All predefined clefs supported by LilyPond with custom offsets
- New translations: Dutch, Swedish, Hebrew, German

Please download and test!

Translators, no new phrases are going to be introduced until the final
0.4 release at the end of this month. Please test the existing
translations and fix them. Deadline to submit translations which will
make it to 0.4 release is 23rd September, 24:00PM CET.


Regards.
- Matevž
