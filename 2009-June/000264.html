Hi,

I have found out why Canorus failed to configure on my system. The reason was 
that I put the sources in a directory called c++ which made something go 
wrong with regular expressions in cmake:

RegularExpression::compile(): Nested *?+.
RegularExpression::compile(): Error in compile.
CMake Error at /usr/share/cmake-2.6/Modules/UseSWIG.cmake:76 (IF):
  if had incorrect arguments: ${swig_source_file_path} MATCHES
  "^${CMAKE_CURRENT_SOURCE_DIR}" (Regular expression
  "^/home/andreas/c++/canorus-0.7svn.R1100/src" cannot compile).
Call Stack (most recent call first):
  /usr/share/cmake-2.6/Modules/UseSWIG.cmake:183 (SWIG_ADD_SOURCE_TO_MODULE)
  src/CMakeLists.txt:544 (SWIG_ADD_MODULE)

Renaming that directory from c++ to src solved the problem.

Cheers

Andreas
