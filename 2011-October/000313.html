Hi,

For all of you following changes to Bugs, Features and/or Patches,
note the following important information:

As soon as these tracker informations have been imported into Sourceforge
you are required to add monitoring to your bug if you wish to follow it.

If I work on a bug I often have additionally questions, so it's important
to have a sourceforge account to make sure your reported bugs get fixed.

Unanswered or not reproducible bugs will get closed after some time (usually
when a new release was published). If this should the case please try to find
the error with the new release again and if it still exists reopen the bug.
