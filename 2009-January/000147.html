Hm... that's wierd. Do you have libutil.so present on your system?
$ locate libutil.so
should return something. On Linux, it's part of libc devel library.

The -lutil flag is set because Canorus python module didn't compile on
Gentoo some time ago. You can comment this from lines 48-51 in
src/CMakeLists.txt and rerun cmake.

Please tell us, if it works so we can apply the patch. We didn't have
any Mac users for some time so we might have broken something.
