Greetings,

The work I've been doing for the last half a year finally showed some
results. A few thousand lines of predicates written in Prolog for
Harmonia project obviously makes sense because it works! :)

Harmonia SVN R35 added Canorus plugin to its mainstream SVN tree. And
Canorus SVN R352 added some additional arguments the harmonia and
possibly other plugins need (like the actual plugin directory, more than
a single argument to the function).

The work is still experimental but it already demonstrates how the
harmony analysis should look like. You can see a nice demonstration here
http://harmonia.berlios.de/files/harmonia.html
.

I've also written down some remarks from the practice usage of Canorus
scripting engine.
https://canorus.berlios.de/wiki/index.php/Plugins

Please take your time, test and tell me what you think!


Regards.
