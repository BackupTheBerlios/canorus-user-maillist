Concerning NoteEdit: Make sure your midi system is up and running
or try disabling it via command line or via build scripts. Are you sure
you installed Canorus from sources? Because glibc is part of the base
system, so this error should not happen except you have a different
build and use platform (like it was built on 32bit and I run it on 64bit).

Except the readme file you can also browse the NoteEdit Mailing list
archive containing several informations about Midi problems.
