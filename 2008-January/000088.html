Canorus development team is happy to announce the new release of Canorus
- a free cross-platform music score editor, version 0.5beta. This
release is meant mainly for translators, testers to track down the bugs and for
developers to fix them  :) 

Download beta at:
http://developer.berlios.de/project/showfiles.php?group_id=6144

Changes:
- new archive file format
- marks
- etc.

New features not fully implemented yet:
- midi export
- printing
- playback of marks
- saving/loading marks

Testers please use bugtracker:
https://developer.berlios.de/bugs/?group_id=6144

Translators, please download the last version of your language (or template.ts, if your language doesn't exist yet) from
http://canorus.jekovec.net/lang
and translate it using Qt Linguist.

The release candidate is planned on 14th January and translations should be done by then.
