QFont is used for locating the font at runtime. It's not possible for
us to guess where a font might be installed. QFontDatabase does not
provide that information. Apparently Linux, Win and Mac all use
different font system and you have to query your OS manual how to
configure your font pathes (or how/where they are configured).

On my system I have in /etc/X11/fs/config the following entry:

catalogue = catalogue:/etc/X11/fontpath.d,
         /usr/share/fonts/msttcorefonts

The directory contains links in some more or less cryptic form
to the font directories. Things like western font type come
into the game as well here... I really haven't looked in xorg
wiki for more detailed information.
