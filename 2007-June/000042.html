> Instructions:
>
> - Start up Canorus
> - Click on Staff
> - Select "Playable" from side tool bar (third icon from top)
> - Enter notes on staff
> - You can select different note lengths from the an icon on the top tool 
> bar.
>
>
> Discussion:
>
> One of the problems with the the current interface is the most important 
> items are not visible at the initial start up. Probably the first thing 
> a new user will ask is "where are my quarter notes?" or half notes, etc.
>
> A simple fix would be to have the top staff selected by default at start 
> up. This would present the user with the most common tools from the 
> beginning.
>
> I do think there are some more changes that can be made to improve 
> usability. I am currently reading Donald A. Norman's classic text on 
> usability, "The Design of Everyday Things." I hope to share some 
> thoughts in the coming weeks about the interface and design. Although 
> I'm not a Canorus developer, I am interested in its development.
