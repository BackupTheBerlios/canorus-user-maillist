Fixes are in the changelog
http://developer.berlios.de/project/shownotes.php?release_id=12899

What should we expect to work?
Well, you're able to insert mulitple voice vocal score with clef, time
sig, key sig, notes, rests, accidentals, all barlines, ties and slurs
and with lyrics. There are also some advanced features like function
markings, scripting stuff, source view in CanoursML and LilyPond syntax etc.

What are we going to see in the future?
Still lots to do - in short: undo, redo, copy, paste, note stuff like
fermata, stacatto, marcatto, dynamics, tempo markings, drum notes,
guitar tablatures, custom text, pedal markings, grace notes, document
layout, printing support, users manual.

See roadmap
https://canorus.berlios.de/wiki/index.php/Roadmap
