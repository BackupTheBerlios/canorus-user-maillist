Hi, Canorus 0.3.1 packages for Mac OS X are available here:

http://merphant.net/canorus/

Actually this is latest SVN, r509.

This is a universal binary package, so it should run on both PPC and
Intel machines.

Unfortunately I have been having problems bundling the Qt and Python
libs, so for now you will have to download and install those
separately; see the README-OSX.txt file in the package, or go to
http://canorus.berlios.de/wiki/index.php/Developers_Guide#Mac_OS_X for
the links to download those files.

Any Mac users out there, please test.

Regards,
