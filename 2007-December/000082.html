On Fri, 14 Dec 2007, canorus-user at lists.berlios.de wrote:

> Back to MUP...I think the included editor Mupmate is pretty
> cool...it's nice to have an editor optimized for the syntax. My main
> issues with using MUP full time for composing are with midi playback.
> As a composer, it would be nice to be able to play back, say, just a
> few measures in the middle of a piece I'm working on, instead of the
> whole thing. I think that would get pretty annoying once I start
> working on longer pieces! Are there any workarounds to this

This is a little off topic ( join the mupusers mailing list ! ) but I'll 
give a short answer anyway: Take a look at the mup commandline options.
The -x option allows you to select just a few measures and then use the 
-m/-M options to create a MIDI file or -f/-F to create a Postscript.
In addition, the -s option allows selecting one or more staffs instead of 
all staffs.

Maybe this is not possible with mupmate. You have to use mup from the 
commandline to accomplish this I think. I haven't used mupmate very much, 
I prefer Vim. (with syntax color highlighting !)
