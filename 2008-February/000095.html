Tomas,

The currently the user documentation is quite limited. The user guide 
contains some information on lyrics and current key bindings:
http://canorus.berlios.de/wiki/index.php/Users_guide

Of the notation software I've tried, Sibelius seems to have the best 
accessibility support among the high end software. Noteworthy Composer 
is a cheaper option that has nice PC keyboard support for note entry.

Canorus is not quite ready for prime time but it is developing quickly. 
Your experience and feedback is especially helpful.

I filed a bug report about the bar lines.
https://developer.berlios.de/bugs/?func=detailbug&bug_id=13071&group_id=6144

A feature request already exists for note entry via PC keyboard.
https://developer.berlios.de/feature/?func=detailfeature&feature_id=2837&group_id=6144

Could you elaborate on your statement, "I'm confused by fact that notes 
are placed exactly where I click"?
