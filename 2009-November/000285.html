Hi Martin and sorry for the late reply.

Lilypond 2.13 uses completely different Emmentaler font than 2.12,
that's why the font isn't compatible with Canorus. One solution that
usually works is to install Lilypond from maintainers website and use
their own paths which don't collide with the system fonts (lily and its
fonts are installed /usr/local/lilypond/). The second solution is
probably that your canorus package actually doesn't contain
Emmentaler-14.ttf file at all. Default search path for Canorus fonts is
/usr/share/canorus/fonts, so only if emmentaler is not there, it tries
to use the system wide font.

I have both Lilypond-2.13 and Canorus SVN trunk installed on my Ubuntu
and the fonts work fine.
